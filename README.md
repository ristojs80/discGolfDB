# About
This is proof of concept database for discgolf. 

Programmed in Python and JavaScript.  
Uses the following Python libraries:  
flask  
pony orm  

Other technologies used:  
BootStarp CSS framework  
SQLite




# How to run

pipenv shell  
pipenv sync  
python app.py
